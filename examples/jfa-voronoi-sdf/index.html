JFA Voronoi + SDF

This demo builds a Voronoi diagram using the Jump Flooding Algorithm (JFA) and derives a distance field (DF/SDF) from it. All computation runs in fragment shaders on the GPU.

Instructions: click and drag on the canvas to move the nearest seed and watch the Voronoi/SDF update in real time. Use the controls to toggle visualization modes, borders, seed count, and isolines.

This example is part of the gpu-io examples. Source code for this standalone example: Github.
