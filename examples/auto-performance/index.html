Auto-Performance Tuning Demo

This demo showcases gpu-io's automatic performance tuning feature. The simulation adapts its quality settings in real-time based on your device's performance capabilities. You can manually override the auto-tuning to compare different quality presets and see how they affect both visual quality and performance.

Auto-Tuning Features:
• Automatically detects device performance capabilities
• Dynamically adjusts quality settings to maintain smooth framerates
• Provides four quality presets: high (highest), medium (medium), low (low), and minimal (minimal)
• Can be enabled/disabled and customized per application needs

Instructions: Use the controls on the right to toggle between automatic and manual quality settings. Touch/mouseover to interact with the fluid simulation. Watch the FPS counter to see how different quality settings affect performance. Press "v" to record a video.

Quality Presets:
• High: Maximum quality with highest particle count and resolution
• Medium: Balanced quality and performance
• Low: Reduced quality for better performance on slower devices
• Minimal: Minimal quality for maximum compatibility

Note for Reduced-Motion Users: If you have reduced-motion preferences enabled, auto-tuning may be disabled by default. You can manually enable it using the controls or by setting respectReducedMotion: false in the configuration.

This page is part of a series of examples using gpu-io, a GPU-accelerated computing library for physics simulations and other mathematical calculations. Find the source code on Github.
